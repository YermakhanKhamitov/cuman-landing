# Cuman.kz Promo Landing

Промо-лендинг для онлайн-сервиса Cuman.kz. Сайт знакомит пользователя с возможностями сервиса: просмотр продукции, изучение услуг, быстрый переход к заявке и контактной форме.

## Скриншот

![Скриншот главной страницы](assets/img/screenshot.png)

## Стек технологий

- HTML5
- CSS3
- Vanilla JavaScript
- Адаптивная верстка
- Flexbox
- CSS Grid

## Функциональность

- Семантическая HTML-структура
- Адаптация под мобильные устройства, планшеты и десктоп
- Бургер-меню для мобильной версии
- Плавный скролл по якорным ссылкам
- Анимация появления блоков при скролле
- Аккордеон в блоке "Как работает"
- Форма заявки с простой обработкой на JavaScript
